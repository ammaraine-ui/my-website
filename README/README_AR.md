# نظام عمار العاني للإدارة والمبيعات

هذه حزمة بداية احترافية لنظام شبكة عربي لإدارة:
- المبيعات ونقطة البيع POS
- المشتريات
- المواد والمخازن
- الحسابات
- العملاء والموردين
- المستخدمين والصلاحيات
- الموظفين والتقارير

## التقنية المقترحة
- WPF + .NET 8
- SQL Server
- Entity Framework Core
- MVVM

## المجلدات
- `Database` : سكربت قاعدة البيانات الأولي
- `Architecture` : هيكل المشروع والموديولات
- `UI` : وصف الشاشات والستايل
- `Source` : ملفات C# و XAML أولية كنقطة انطلاق

## ملاحظات
هذه الحزمة **تصميم وبداية تطوير احترافية** وليست النسخة النهائية الكاملة الجاهزة للتشغيل.
الخطوة التالية تكون تنفيذ المشروع فعلياً على Visual Studio وربط الشاشات مع قاعدة البيانات.
