- الموردون
- المستخدمون
- الأدوار والصلاحيات
- التقارير
- نقطة بيع أسرع مع مواد سريعة وبحث بالباركود/الكود

1) ارفع المشروع كاملًا مع نفس المجلدات.
2) أضف Vercel Postgres من لوحة المشروع.
3) انشر المشروع.
4) افتح الموقع واضغط "تهيئة قاعدة البيانات" مرة واحدة.
5) سجل دخول بالحساب الافتراضي:
